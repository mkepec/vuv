## 🗓️ Vaš 15-Tjedni Journey

### 🏗️ **Tjedni 1-3: Culture Shock & Foundation**
*"Dobrodošli u stvarni svijet software development-a"*

- **Tjedan 1:** Prvi dan u VelocityTech - svjedočite production disaster-u
- **Tjedan 2:** Git chaos - "Gdje mi je kod?!" scenario  
- **Tjedan 3:** Ana's testing nightmare - zašto QA postaje bottleneck

**Učit ćete:** DevOps principi, Git workflows, test automation osnove

### ⚙️ **Tjedni 4-6: Technical Revolution**  
*"It works on my machine" → "It works everywhere"*

- **Tjedan 4:** Build mayhem - dependency hell i environment chaos
- **Tjedan 5:** Container revolution - Docker rješava compatibility probleme
- **Tjedan 6:** Orchestration magic - Kubernetes za scale-ability

**Učit ćete:** CI/CD pipeline-e, Docker, Kubernetes osnove

### 🏗️ **Tjedni 7-9: Infrastructure Transformation**
*"Od snowflake server-a do Infrastructure as Code"*

- **Tjedan 7:** Filip's infrastructure nightmare - ručno setup server-a
- **Tjedan 8:** Configuration chaos - svaki server je drugačiji  
- **Tjedan 9:** Security wake-up call - DevSecOps integration

**Učit ćete:** Terraform, Ansible, sigurnosne prakse

### 👁️ **Tjedni 10-12: Observability & Reliability**
*"Od gašenja požara do predviđanja problema"*

- **Tjedan 10:** Network complexity - mikroservisi ne mogu komunicirati
- **Tjedan 11:** "Što se događa u production?" - monitoring setup
- **Tjedan 12:** "Igli u plastu sijena" - centralized logging

**Učit ćete:** Kubernetes networking, Prometheus/Grafana, log management

### 🎯 **Tjedni 13-15: Capstone Transformation**
*"Sve povezano - vaš vlastiti DevOps masterpiece"*

- **Tjedan 13:** Agile planning - kako organizirati veliku transformaciju
- **Tjedan 14:** Implementation sprint - svatko postane ekspert za svoju domenu
- **Tjedan 15:** Executive showcase - prezentirate rezultate transformacije

**Stvorit ćete:** Kompletnu DevOps infrastrukturu kao portfolio projekt
